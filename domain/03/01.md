这些 basic types 对应的 JDBC type/Java type 以及在 org.hibernate.type.BasicTypeRegistry 中注册的 key(短名)参见原表


要使用 hibernate-spatial types 需要添加 hibernate-spatial 依赖并使用 org.hibernate.spatial.SpatialDialect 的实现类。
